# User Manager

Experiments during beach time with Java Spring.

## Getting Started

### user-manager-spring

#### Dependencies
- Java
- Postgresql

#### Setup steps
- Setup the database
`make refreshdb`

- start the app
`make start`

- run the tests
`make tests`
